# אפליקציית מנהל משימות

אפליקציית Todo פשוטה שנבנתה ב-React ומאפשרת ניהול משימות יומי.

## הרצה מקומית
npm install
npm run dev

## קומפוננטות
- App – ניהול state ולוגיקה כללית
- TodoInput – הוספת משימות
- TodoList – הצגת רשימת משימות
- TodoItem – משימה בודדת (עריכה/מחיקה/סימון)
- FilterButtons – סינון משימות
